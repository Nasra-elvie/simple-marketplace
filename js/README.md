# Simple Online Marketplace (Frontend Capstone Project)

## Overview
This project is a simple online marketplace frontend built with **HTML, CSS, and JavaScript**.  
It demonstrates responsive design, product listing with search/filter, and a user registration form with validation and cookie setup.

## Features
- **Homepage:** Product cards grid with images, names, prices, and "View Details" buttons.
- **Product Listing:** Search bar and category filter (fruits/vegetables).
- **Registration Form:** Username, email, and password validation. Stores username in a cookie.
- **Privacy Notice:** Compliance with Kenya’s Data Protection Act (2019).

## File Structure
- `index.html` → Homepage
- `products.html` → Product listing page
- `register.html` → Registration form
- `css/styles.css` → Styling
- `js/search.js` → Search/filter logic
- `js/register.js` → Registration validation
- `images/` → Product images
- `docs/documentation.pdf` → 1-page documentation
